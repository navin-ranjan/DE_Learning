# Python for Data Engineering

## Core Theory Concepts (Must Know)
- Python execution model (source -> bytecode -> Python Virtual Machine)
- Object model, mutability, and reference semantics
- Namespaces and scope resolution (LEGB rule)
- Iterables vs iterators vs generators and lazy evaluation
- Concurrency theory: GIL, threads vs processes, async I/O tradeoffs
- Memory management fundamentals (reference counting and garbage collection)
- Packaging and module resolution (`PYTHONPATH`, imports, virtual environments)

## Core Python Fundamentals
- Variables, Data Types & Operators
- Control Flow (if/else, loops)
- Functions & Lambda Expressions
- List, Dictionary, Set, Tuple Comprehensions
- String Manipulation & Formatting
- File Handling (read, write, append)

## Object-Oriented Programming (OOP)
- Classes & Objects
- Inheritance & Polymorphism
- Encapsulation & Abstraction
- Magic Methods (__init__, __str__, __repr__)
- Decorators & Property

## Advanced Python
- Generators & Iterators
- Context Managers (with statement)
- Exception Handling & Custom Exceptions
- Regular Expressions (re module)
- Multithreading & Multiprocessing
- Async Programming (asyncio)

## Data Engineering Libraries
- Pandas (DataFrames, Series, IO operations)
- NumPy (Arrays, Mathematical operations)
- datetime & time modules
- json & csv modules
- os & pathlib for file system operations
- logging module
- pyarrow basics (Arrow memory format and parquet interoperability)
