# Azure Data Fundamentals (DP-900) Exam Preparation

## Exam Syllabus / Topics

### Domain 1: Core Data Concepts (20-25%)
- Data types: Structured, Semi-structured, Unstructured
- Data formats: CSV, JSON, Parquet, Avro
- Data storage solutions
- Data processing: Batch vs Streaming
- OLTP vs OLAP
- Data relationships

### Domain 2: Relational Data on Azure (25-30%)
- Azure SQL Database
- Azure Database for PostgreSQL
- Azure Database for MySQL
- SQL basics (SELECT, JOIN, GROUP BY, WHERE)
- Azure SQL deployment options

### Domain 3: Non-Relational Data on Azure (20-25%)
- Azure Cosmos DB
- Azure Blob Storage
- Azure Table Storage
- Azure File Storage
- Cosmos DB APIs (SQL, MongoDB, Cassandra, Gremlin, Table)

### Domain 4: Analytics Workload on Azure (25-30%)
- Azure Synapse Analytics
- Azure Databricks
- Azure Stream Analytics
- Azure Data Lake
- Power BI basics
